# Hardware

1. MKS TMC2100 ステッピングモータードライバ　（絶版なので、後継にTMC2130など採用ください）
2. SY35ST26-0284A ステッピングモータ
3. aitendo　フォトインタラプタセンサモジュール [PIR393-5V]
4. ArduinoProMini(5V)
